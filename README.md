### 运行

```sh
git clone https://github.com/ChinaGAI/AICHAT
cd ai-chat
npm install
npm run dev
```

### 配置

```
NEXT_PUBLIC_API_HOST=https://api.chat.aieo.cn # api地址
NEXT_PUBLIC_WS_HOST=wss://api.chat.aieo.cn/wss # websocket地址
NEXT_PUBLIC_APP_PREFIX=AI_CHAT # 应用前缀
```
